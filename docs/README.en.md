## 🌟 Getting Started

This repository provides a sandbox for GitHub App development. It offers an environment for safely developing, testing, and refining GitHub Apps without impacting production environments.

## 🚀 How to Start

1. **Clone the repository**:
   ```
   git clone https://github.com/Sunwood-ai-labs/github-app-sandbox.git
   cd github-app-sandbox
   ```

2. **Install dependencies**:
   ```
   pip install -r requirements.txt
   ```

3. **Configure environment**:
   - Copy the `.env.example` file to `.env` and fill in the necessary information.
   ```
   cp .env.example .env
   ```

4. **Create a GitHub App**:
   - Create a new GitHub App in GitHub's [Developer Settings](https://github.com/settings/developers).
   - Set the necessary permissions and specify the Webhook URL.
   - Download the generated private key and store it in a secure location.

5. **Run the sandbox**:
   - Follow the instructions in each sandbox project's README to run the application.

For detailed instructions, please refer to the README for each sandbox project.
